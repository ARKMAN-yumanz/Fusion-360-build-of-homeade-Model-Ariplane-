# Fusion-360-build-of-homeade-Model-Ariplane-
the fusion 360 file for my homemade model airplane 
This file above is my CAD design for my first ever model plane. It is simply a what one may call a by-plane with tis engine(motor with propeller) being in the front of the plane. Providing it with all its thrust. This is my first ever project and I wanted to do something I always found interesting since I was a kid. Airplanes. Therefore, I decided to try making my own plane from scratch. I always marveled and watched youtubers such as Peter Sripol make cool RC planes and I thought of trying it out myself. My dreas are represented in this cad design, which is in my opinion my ideal and realistic design of my first homeade RC plane. I would like this build to inspire and help others who dont know where to start and want to get into aerospace or aerodynamics as well. Following the work I have included in my project and trying to replicate it themselves.
